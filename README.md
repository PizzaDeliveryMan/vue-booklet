# vue-mybook
Vue book-like compoment for create a your own e-book

## Demo
Readying

## Installation
```bash
npm install vue-mybook --save
```

or

```bash
yarn add vue-mybook
```

## Usage

Import module from days-calendar, then register it to Vue:


```js
import vueMybook from 'vue-mybook';

new Vue({
  el: '#your-dom-element',
   components: {
      vueMybook
   }
})
```

## Build Setup
